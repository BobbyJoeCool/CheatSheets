CSS · Sheet 10

Colors

color · opacity · hex · rgb · hsl · rgba · hsla · currentColor

Description

CSS colors can be written in several formats: named keywords, hexadecimal, rgb(), and hsl(). The hsl() format — hue, saturation, lightness — is the most human-readable for building and adjusting palettes. Modern CSS unifies rgb/rgba and hsl/hsla into single functions with an optional slash-separated alpha. opacity affects the entire element including its children; an alpha channel in a color value affects only that single property.

Example

/* Named keyword */
p { color: tomato; }

/* Hex: #RRGGBB — or #RGB shorthand */
h1 { color: #5c6bc0; }
h1 { color: #59c; }             /* expands to #5599cc */

/* RGB — legacy and modern */
.a { color: rgb(92, 107, 192); }
.b { color: rgb(92 107 192 / 0.8); }  /* modern + alpha */

/* HSL — hue 0-360, saturation %, lightness % */
.c { color: hsl(231, 44%, 56%); }
.d { color: hsl(231 44% 56% / 0.5); } /* modern + alpha */

/* opacity vs alpha channel */
.whole   { opacity: 0.5; }             /* element + ALL children */
.precise { color: rgb(0 0 0 / 0.5); } /* only this color property */

/* currentColor — inherits computed color value */
.icon svg { fill: currentColor; }
.btn      { border: 2px solid currentColor; }

Notes

  • Modern syntax uses spaces and a slash for alpha: rgb(0 0 0 / 0.5) and hsl(240 100% 50% / 0.5)
  • opacity: 0.5 makes the element and all its children semi-transparent; color alpha only affects that property
  • currentColor inherits the element's computed color value — ideal for SVG fills and matching borders
  • CSS Color Level 4 adds oklch() for perceptually uniform color spaces that support wide-gamut displays

Code Explained

color: tomato — named keyword; ~140 named colors exist
#5c6bc0 — hex: two digits per channel (RR GG BB) in base-16
#59c — three-digit shorthand: each digit is doubled
rgb(92 107 192 / 0.8) — modern syntax: channels then alpha after /
hsl(231 44% 56%) — hue 0–360°, saturation %, lightness %
opacity: 0.5 — entire element and all descendants at 50%
rgb(0 0 0 / 0.5) — only that color declaration is semi-transparent
currentColor — inherits the element's computed color value

⚠ Common Mistakes

Using opacity: 0.5 when only the background should be transparent — text fades too
Writing raw hex values without custom properties — impossible to update consistently
Using old comma syntax hsl(H, S%, L%) in new code — prefer the modern space syntax
Assuming #000 and #000000 differ — they are identical

✓ Tips & Best Practices

Use hsl() for palettes — adjust lightness to create tints (80%) and shades (30%) of one hue
Store all colours as custom properties: --color-primary: hsl(231 44% 56%)
Use currentColor for SVG fills and borders that must always match the text colour
Target WCAG AA contrast: at least 4.5:1 for normal text, 3:1 for large text

Quick Summary

Formats: keyword · #hex · rgb() · hsl() — all accept an optional alpha
HSL is most intuitive for palette building — tweak lightness to make tints and shades
opacity affects the whole element; color alpha affects only that one property
currentColor inherits the element's color — great for SVG and border consistency
