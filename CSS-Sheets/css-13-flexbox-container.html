Flexbox: Container

display:flex · flex-direction · flex-wrap · justify-content · align-items · gap

Description

Flexbox is a one-dimensional layout system that arranges direct children along a single axis — row or column. display: flex on a container makes its children flex items. The main axis is defined by flex-direction; the cross axis is perpendicular. justify-content distributes items along the main axis; align-items aligns them on the cross axis. gap replaces old margin hacks for consistent spacing between items.

Example

.container {
  display: flex;
  flex-direction: row;  /* row | column | row-reverse | column-reverse */
  flex-wrap: wrap;      /* nowrap | wrap | wrap-reverse */
  gap: 1rem;            /* uniform gap between all items */
  gap: 1rem 2rem;       /* row-gap | column-gap */
}

/* Main-axis alignment */
.a { justify-content: flex-start; }   /* default */

/* Cross-axis alignment */
.e { align-items: stretch; }          /* default */

/* Classic centering pattern */
.center {
  display: flex;
  justify-content: center;
  align-items: center;
}

/* Nav: logo left, links right */
.nav {
  display: flex;
  justify-content: space-between;
  align-items: center;
}

Notes

  • Flexbox is one-dimensional — use Grid when you need both rows and columns simultaneously
  • gap adds space between items only, not before the first or after the last
  • flex-wrap: wrap lets items flow onto new lines; without it items shrink to stay on one line
  • align-content aligns multiple wrapped lines; align-items aligns within each line

Code Explained

display: flex — block-level flex container; children become flex items
flex-direction: row — main axis is horizontal (default)
flex-direction: column — main axis is vertical
flex-wrap: wrap — items wrap onto new lines when they overflow the container
gap: 1rem — uniform spacing between all items with no edge space
justify-content — alignment along the main axis
align-items: center — centres items on the cross axis
justify-content: center + align-items: center — centres anything

justify-content

Syntax Description
flex-start Pack to start (default)
flex-end Pack to end
center Centre along main axis
space-between Equal space between items
space-around Equal space around items
space-evenly Equal space everywhere

align-items

Syntax Description
stretch Fill cross axis (default)
flex-start Align to cross-start
flex-end Align to cross-end
center Centre on cross axis
baseline Align text baselines

flex-direction

Syntax Description
row L → R, horizontal (default)
row-reverse R → L, horizontal
column T → B, vertical
column-reverse B → T, vertical

⚠ Common Mistakes

  • Using margins instead of gap — margins add unwanted space before first and after last item
  • Applying align-items expecting multi-line behaviour — align-content handles multiple lines
  • Assuming justify-content is always horizontal — it follows the main axis, which changes with flex-direction
  • Setting flex properties on items instead of the container — container properties must go on the parent

✓ Tips & Best Practices

  • display:flex; justify-content:center; align-items:center — the universal one-liner centering pattern
  • margin-left: auto on a flex item pushes it and everything after it to the far end of the main axis
  • Prefer gap over margins between items — no edge-case spacing and works in both directions
  • flex-direction: column on the body or app wrapper creates a sticky-footer layout

Quick Summary

  • display: flex on the container; direct children become flex items
  • flex-direction sets main axis; justify-content distributes along it
  • align-items aligns on cross axis; align-content aligns multiple wrapped lines
  • gap is the clean modern way to space items — no margin hacks needed
