Gradients

linear-gradient · radial-gradient · conic-gradient · stops · repeating
Example
/* Linear */
.a { background: linear-gradient(to right, #5c6bc0, #9c8fff); }
.b { background: linear-gradient(135deg, #5c6bc0 0%, #9c8fff 100%); }

/* Multi-stop */
.rainbow {
  background: linear-gradient(
    to right, #e57373, #ffb74d, #81c784, #64b5f6, #9c8fff
  );
}

/* Hard stop — no blend at 50% */
.stripe { background: linear-gradient(to right, #5c6bc0 50%, #1c2030 50%); }

/* Radial */
.glow {
  background: radial-gradient(circle at center, #9c8fff 0%, #0f1117 70%);
}

/* Conic — pie chart slices */
.pie {
  background: conic-gradient(
    #e57373 0% 33%, #81c784 33% 66%, #64b5f6 66% 100%
  );
}

/* Repeating stripes */
.stripes {
  background: repeating-linear-gradient(
    45deg,
    #5c6bc0 0px,  #5c6bc0 10px,
    #1c2030 10px, #1c2030 20px
  );
}
Description
CSS gradients are <image> values used anywhere background-image is accepted. linear-gradient() transitions colours along a straight line; radial-gradient() radiates from a centre point; conic-gradient() sweeps around a point like a pie chart. Colour stops define where each colour starts and ends — explicit positions give precise control. Hard stops — two colours at the same position — create sharp edges with zero blending.
Notes
Gradients are images — they work in background, border-image, and mask, not in color
Without explicit stop positions, colours are evenly distributed along the gradient axis
Hard stops (same position twice) create crisp colour changes — great for stripes and sections
Gradients are resolution-independent — they render sharp at any size unlike raster images
Code Explained
to right — direction keyword; degrees also work (135deg)
0%, 100% — explicit stop positions along the gradient axis
Hard stop: same position twice — instant colour change, no blend
radial-gradient(circle at center) — circular radiate from a point
conic-gradient — sweeps around a centre point; ideal for pie charts
red 20% 40% — colour holds solid from 20% to 40%
repeating-linear-gradient — tiles the defined pattern endlessly
transparent as a stop fades the gradient to nothing

⚠ Common Mistakes
Using a gradient as a color value — gradients are images, not colours
Omitting explicit stop positions — browser distribution may not match your intent
Forgetting -webkit-background-clip: text alongside background-clip: text for Safari
Repeating gradients where stops don't cleanly tile — causes visible seam at the repeat boundary
✓ Tips & Best Practices
linear-gradient(to bottom, color, transparent) — classic fade-out overlay on images
Hard stops create crisp CSS stripes — useful for dividers, progress bars, and colour blocks
Conic gradients make pure-CSS pie charts — set stop percentages to slice proportions
Store reusable gradients as custom properties: --gradient-brand: linear-gradient(...)
