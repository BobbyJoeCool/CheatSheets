Typography: Text Properties

text-align · text-decoration · text-transform · letter-spacing · white-space
Description
Text properties control how characters and lines are arranged, decorated, and spaced independently of the font itself. text-align sets horizontal alignment within a block. line-height and letter-spacing tune readability and visual rhythm. The classic truncation pattern — white-space: nowrap + overflow: hidden + text-overflow: ellipsis — clips overflowing text to a single line with an ellipsis.
Notes
Unitless line-height (e.g. 1.6) is preferred — it scales with the element's own font-size
text-align: justify creates uneven word spacing — use sparingly and only for long-form prose
white-space: nowrap prevents wrapping; pre preserves all whitespace as written
text-decoration is a shorthand accepting line type, style, color, and thickness
Code Explained
text-align — left (default LTR), center, right, justify
text-decoration: none — removes underline from links
underline wavy #e57373 — style and color in the shorthand
text-transform: uppercase — CSS transform, HTML content unchanged
letter-spacing: -0.03em — negative tightens; positive loosens
line-height: 1.6 — unitless, relative to the element's own font-size
white-space: nowrap — prevents line breaks inside the element
text-overflow: ellipsis — shows … when clipped (needs overflow:hidden + nowrap)
Example
/* Alignment */
.left    { text-align: left; }    /* default for LTR */
.center  { text-align: center; }
.right   { text-align: right; }
.justify { text-align: justify; }

/* Decoration */
a            { text-decoration: none; }
.strike      { text-decoration: line-through; }
.underline   { text-decoration: underline; }
.fancy       { text-decoration: underline wavy #e57373; }
.thick-under { text-decoration: underline 2px; }

/* Transform — applied via CSS, HTML unchanged */
.upper { text-transform: uppercase; }
.lower { text-transform: lowercase; }
.caps  { text-transform: capitalize; }

/* Spacing */
h1  { letter-spacing: -0.03em; }   /* tight heading */
.label { letter-spacing: 0.12em; } /* airy uppercase label */
p   { line-height: 1.6; }          /* unitless preferred */
p   { word-spacing: 0.05em; }

/* Truncation recipe — all three required */
.truncate {
  white-space: nowrap;
  overflow: hidden;
  text-overflow: ellipsis;          /* shows … at clip point */
}

⚠ Common Mistakes
Using text-transform in HTML attributes — if design changes, markup must change too; use CSS
Setting text-overflow: ellipsis alone — requires overflow: hidden + white-space: nowrap
Using line-height: 1.6px with a unit — unitless scales with font-size; px does not
Applying letter-spacing to body text — best reserved for headings and uppercase UI labels
✓ Tips & Best Practices
Use unitless line-height: 1.5–1.7 for body, 1.1–1.2 for headings
Negative letter-spacing (-0.02em to -0.04em) on large headings looks more intentional and refined
Memorise the truncation recipe — it's one of the most commonly needed CSS patterns
Use text-wrap: balance (modern) on headings to avoid awkward single-word last lines
Quick Summary
text-align: left | center | right | justify | start | end
text-decoration: none | underline | line-through — accepts style and color
Truncation: white-space: nowrap + overflow: hidden + text-overflow: ellipsis
Use unitless line-height — it scales relative to the element's own font-size
