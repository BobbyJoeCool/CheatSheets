Emphasis & Inline

Syntax Renders as Support
**text** or __text__ bold All
*text* or _text_ italic All
***text*** bold italic All
~~text~~ strikethrough GFM / Obsidian
==text== highlight Obsidian
`code` inline All

Links & Images

Syntax Notes
[text](url) Inline link
[text](url "title") Link with tooltip
[text][id] … [id]: url Reference-style link
<url> Auto-link (GFM bare URLs also work)
![alt](url) Image
![[file.png]] Obsidian internal embed
[[Page Name]] Obsidian wikilink

Flavor Notes

  • CommonMark — strict spec; base for most renderers
  • GFM (GitHub Flavored) — adds tables, task lists, ~~strikethrough~~, autolinks, fenced code
  • Obsidian — GFM + ==highlight==, ![[embeds]], [[wikilinks]], ^block-refs, callouts
  • Blank line between paragraphs is required — single newlines are joined
  • Two trailing spaces or \ at line end forces a line break
  • Indent nested lists with 2–4 spaces consistently
  • Escape special characters with \ — works on: * _ ` [ ] ( ) # + - . ! |
