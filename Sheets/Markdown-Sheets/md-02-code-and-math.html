Math / LaTeX Syntax

<!-- ── Inline math ───────────────────────────── -->
$E = mc^2$                     ← inline (single dollar)
\(E = mc^2\)                   ← inline (LaTeX delimiter)
<!-- ── Display / block math ──────────────────── -->
$$
E = mc^2
$$

\[
\int_0^\infty e^{-x^2} dx = \frac{\sqrt{\pi}}{2}
\]
<!-- ── Obsidian: also supports inline blocks ─── -->
$$E = mc^2$$                   ← single-line display block
<!-- ── Aligned equations (amsmath) ──────────── -->
$$
\begin{align}
  a &= b + c \\
  x &= y - z
\end{align}
$$
<!-- ── Matrix ─────────────────────────────────── -->
$$
\begin{pmatrix}
  a & b \\
  c & d
\end{pmatrix}
$$
<!-- ── Cases ─────────────────────────────────── -->
$$
f(x) = \begin{cases}
  x^2 & \text{if } x \geq 0 \\
  -x  & \text{if } x < 0
\end{cases}
$$
