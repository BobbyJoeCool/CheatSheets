Set Examples

// ── Creating a Set ────────────────────────────────────────────
const s = new Set([1, 2, 3, 2, 1]);  // duplicates removed
s.size;                 // 3
[...s];                 // [1, 2, 3]

// ── CRUD ──────────────────────────────────────────────────────
s.add(4);               // Set { 1, 2, 3, 4 }
s.add(2);               // no change — already exists
s.has(3);               // true
s.has(99);              // false
s.delete(1);            // true — removed
s.size;                 // 3

// ── Deduplication — most common use ──────────────────────────
const arr = [1, 2, 2, 3, 3, 3, 4];
const unique = [...new Set(arr)];    // [1, 2, 3, 4]

const tags = ["js", "ts", "js", "node", "ts"];
const uniqueTags = Array.from(new Set(tags)); // ["js","ts","node"]

// ── Iteration ─────────────────────────────────────────────────
for (const val of s) {
  console.log(val);
}
s.forEach(val => console.log(val));

// ── Set operations (ES2024+ native, or manual) ────────────────
const a = new Set([1, 2, 3, 4]);
const b = new Set([3, 4, 5, 6]);

// Native (ES2024)
a.union(b);             // Set {1,2,3,4,5,6}
a.intersection(b);      // Set {3,4}
a.difference(b);        // Set {1,2}
a.symmetricDifference(b); // Set {1,2,5,6}
a.isSubsetOf(b);        // false
a.isSupersetOf(b);      // false

// Manual (pre-ES2024)
const union    = new Set([...a, ...b]);
const intersect = new Set([...a].filter(x => b.has(x)));
const diff     = new Set([...a].filter(x => !b.has(x)));

// ── Fast membership check ─────────────────────────────────────
const validRoles = new Set(["admin", "editor", "viewer"]);
function isValidRole(role) {
  return validRoles.has(role);  // O(1) vs O(n) array includes
}

// ── WeakSet — object values only; not iterable ────────────────
const ws = new WeakSet();
const obj = { id: 1 };
ws.add(obj);
ws.has(obj);            // true
ws.delete(obj);

// Use case: track processed objects without preventing GC
const processed = new WeakSet();
function process(item) {
  if (processed.has(item)) return;
  processed.add(item);
  // ... do work
}

// ── TypeScript typed Set ──────────────────────────────────────
const ids: Set<number> = new Set([1, 2, 3]);
ids.add(4);
ids.add("five"); // ✖ Error: string not assignable to number

Notes

Set uses SameValueZero equality — same as === except NaN === NaN (so only one NaN can be stored)
.has() is O(1) — much faster than Array.includes() which is O(n); use Set when membership checks are frequent
ES2024 native set methods (union, intersection, etc.) — available in Node 22+ and modern browsers; polyfill for older environments
To use map/filter on a Set, convert to array first: [...set].filter(...)
WeakSet — values must be objects; GC-friendly; useful for tracking which objects have been processed without leaking memory
Object values in Sets use reference equality — two objects with the same shape are different entries

Common Mistakes

  • Trying to deduplicate objects with Set — objects use reference equality, so new Set([{a:1},{a:1}]) has two entries
  • Accessing Set values by index — Sets have no index; convert to array with spread or Array.from
  • Iterating a WeakSet — it is not iterable; use a regular Set if you need iteration
  • Using Array.includes() in a hot loop instead of a Set lookup — O(n) vs O(1) can matter significantly

Tips

  • [...new Set(arr)] — fastest one-liner to deduplicate a primitive array
  • Convert allowed-values lists to Sets for fast membership checks: const VALID = new Set(["a","b","c"])
  • Use ES2024 set methods (a.intersection(b)) when available — much cleaner than the manual spread + filter approach

Summary

  • Set — unique values · insertion-ordered · .add/.has/.delete/.size
  • .has() is O(1) — far faster than Array.includes() for membership checks
  • ES2024: .union/.intersection/.difference/.symmetricDifference native set operations
  • WeakSet — object values only · GC-friendly · not iterable
