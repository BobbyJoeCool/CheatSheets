Examples

const str = "Hello, World!";

// ── Extracting ───────────────────────────────────────────────
str.slice(7, 12)         // "World"
str.slice(-6)            // "orld!"  (6 from end)
str.slice(0, 5)          // "Hello"
str.substring(7, 12)     // "World"

// ── Searching ────────────────────────────────────────────────
str.includes("World")    // true
str.startsWith("Hello")  // true
str.endsWith("!")        // true
str.indexOf("o")         // 4   (first o)
str.lastIndexOf("o")     // 8   (last o)

// ── Case ─────────────────────────────────────────────────────
str.toUpperCase()        // "HELLO, WORLD!"
str.toLowerCase()        // "hello, world!"

// ── Trimming ─────────────────────────────────────────────────
"  hello  ".trim()       // "hello"
"  hello  ".trimStart()  // "hello  "
"  hello  ".trimEnd()    // "  hello"

// ── Splitting ────────────────────────────────────────────────
"a,b,c".split(",")       // ["a", "b", "c"]
"hello".split("")        // ["h","e","l","l","o"]
"a  b".split(/\s+/)      // ["a", "b"]  (regex sep)
"a,b,c".split(",", 2)    // ["a", "b"]  (limit)

// ── Replacing ────────────────────────────────────────────────
"foo foo".replace("foo", "bar")    // "bar foo"  (first only)
"foo foo".replaceAll("foo", "bar") // "bar bar"
"hello".replace(/[aeiou]/g, "*")   // "h*ll*"   (regex)

// ── Padding ──────────────────────────────────────────────────
"5".padStart(3, "0")     // "005"
"hi".padEnd(5, ".")      // "hi..."
"42".padStart(5)         // "   42"  (default pad is space)

// ── Other ────────────────────────────────────────────────────
"ha".repeat(3)           // "hahaha"
["a","b","c"].join("-")  // "a-b-c"  (Array→string)

// Locale-aware sort
["ä", "z", "a"].sort((a,b) => a.localeCompare(b));
// ["a", "ä", "z"]  (correct for German locale)

Notes

.slice(start, end) — end is exclusive; negative indices count from end; preferred over substring
.split("") — splits into individual characters, but breaks on emoji (use [...str] for Unicode safety)
.replace(string, rep) — only replaces the first occurrence; use .replaceAll() or a /regex/g for all
.padStart(n, "0") — essential for formatting numbers with leading zeros (dates, IDs)
.match(/regex/g) returns an array of all matches or null — always check for null before using
.localeCompare() — use instead of </> when sorting strings for human display

Common Mistakes

  • Using .replace("x", "y") expecting all occurrences to change — only replaces the first
  • Sorting strings with .sort() without a comparator — uses Unicode order, not natural alphabetical
  • Using .split("") on strings with emoji — splits into surrogates; use [...str] instead
  • Forgetting .match() returns null when there is no match — always check before destructuring

Tips

  • Chain methods cleanly: input.trim().toLowerCase().replace(/\s+/g, "-")
  • Use .padStart(2, "0") to format date/time parts: String(hours).padStart(2,"0")
  • Use .matchAll() over repeated .match() calls when you need all matches with capture groups

Summary

  • All string methods return a new string — originals are never modified
  • .slice · .includes · .split · .trim · .replace — the most-used five
  • .replaceAll() for all occurrences; .replace(/pat/g) also replaces all with regex
  • .localeCompare() for correct human-readable string sorting
