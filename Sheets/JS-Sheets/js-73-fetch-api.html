Fetch Examples

// ── Basic GET ─────────────────────────────────────────────────
const res  = await fetch("https://api.example.com/users");
const data = await res.json();

// ── Always check res.ok — fetch only rejects on network error! ─
async function getUser(id) {
  const res = await fetch(`/api/users/${id}`);
  if (!res.ok) {
    throw new Error(`HTTP ${res.status}: ${res.statusText}`);
  }
  return res.json();
}

// ── POST with JSON body ────────────────────────────────────────
const res2 = await fetch("/api/users", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({ name: "Alice", email: "a@b.com" }),
});
const created = await res2.json();

// ── PUT / PATCH / DELETE ──────────────────────────────────────
await fetch(`/api/users/${id}`, {
  method: "PUT",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify(updatedUser),

await fetch(`/api/users/${id}`, { method: "DELETE" });

// ── Auth header ───────────────────────────────────────────────
const res3 = await fetch("/api/protected", {
  headers: {

// ── AbortController — cancel a request ───────────────────────
const controller = new AbortController();
const timeoutId  = setTimeout(() => controller.abort(), 5000); // 5s timeout

try {
  const res4 = await fetch("/api/slow", { signal: controller.signal });
  clearTimeout(timeoutId);
  return await res4.json();
} catch (err) {
  if (err.name === "AbortError") {
    console.error("Request timed out");
  } else {
    throw err;
  }
}

// ── Helper wrapper ────────────────────────────────────────────
async function apiFetch(url, options = {}) {
  const res = await fetch(url, {
    headers: { "Content-Type": "application/json", ...options.headers },
    ...options,
  });
  if (!res.ok) {
    const error = await res.json().catch(() => ({}));
    throw Object.assign(new Error(error.message ?? res.statusText), {
      status: res.status,
    });
  }
  return res.json();
}

// ── TypeScript: typed fetch ───────────────────────────────────
interface User { id: number; name: string; email: string; }

async function fetchUser(id: number): Promise<User> {
  const res = await fetch(`/api/users/${id}`);
  if (!res.ok) throw new Error(`HTTP ${res.status}`);
  return res.json() as Promise<User>;
}

Notes

fetch only rejects on network failure — a 404 or 500 response resolves successfully; always check res.ok
The response body can only be read once — calling .json() consumes the stream; use res.clone() if you need to read it twice
AbortController — cancel in-flight requests on timeout, component unmount, or user navigation; prevents stale responses updating state
Content-Type: application/json header is required for POST/PUT — without it many servers won't parse the body
fetch is available natively in browsers and Node 18+ — no polyfill needed in modern environments
TS: res.json() returns Promise<any> — always assert or validate the type; use zod for untrusted API responses

Common Mistakes

  • Not checking res.ok — a 404 doesn't throw; code silently processes an error response as success
  • Reading the body multiple times — throws "body already used"; clone first if needed
  • Forgetting Content-Type: application/json on POST — server receives the body as plain text or rejects it
  • Not handling AbortError separately — it gets caught by generic error handlers and logged as an unexpected error

Tips

  • Create a wrapper function (apiFetch) that handles res.ok checking, headers, and error parsing — reuse it everywhere instead of raw fetch
  • Always attach an AbortController signal with a timeout for external API calls — prevents hanging requests from blocking your server
  • In React, cancel fetch requests when a component unmounts using AbortController in a useEffect cleanup function

Summary

  • fetch rejects only on network error — always check res.ok for HTTP errors
  • res.json() · text() · blob() · arrayBuffer() — read body once; clone to read twice
  • AbortController — cancel requests on timeout or unmount
  • POST/PUT: set Content-Type: application/json + JSON.stringify(body)
