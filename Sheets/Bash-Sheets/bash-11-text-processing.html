Text Processing Commands

# ── sed — stream editor ───────────────────────────────────────
sed 's/old/new/' file.txt          # replace first match per line
sed 's/old/new/g' file.txt         # replace all matches (global)
sed 's/old/new/gI' file.txt        # case-insensitive replace
sed -i '' 's/foo/bar/g' file.txt   # in-place edit (macOS: -i '' required)
sed -n '5,10p' file.txt            # print lines 5–10 only
sed '/^#/d' file.txt               # delete comment lines
sed 's/^[[:space:]]*//' file.txt   # strip leading whitespace

# ── awk — pattern / field processor ──────────────────────────
awk '{print $1}' file.txt          # print first field (whitespace-delim)
awk '{print $NF}' file.txt         # print last field
awk -F: '{print $1}' /etc/passwd   # use : as delimiter
awk '{sum += $1} END {print sum}'  # sum a column
awk 'NR==5' file.txt               # print line 5 only
awk 'NR>=5 && NR<=10' file.txt     # print lines 5–10
awk '/error/ {print $0}' log.txt   # lines matching pattern
awk '{print NR": "$0}' file.txt    # number every line
awk -F, '{print $2, $4}' data.csv  # CSV: print cols 2 and 4

# ── cut — extract columns ────────────────────────────────────
cut -d: -f1 /etc/passwd            # field 1 with : delimiter
cut -d, -f2,4 data.csv             # fields 2 and 4 (CSV)
cut -c1-10 file.txt                # first 10 characters of each line

# ── sort ─────────────────────────────────────────────────────
sort file.txt                      # alphabetical sort
sort -n numbers.txt                # numeric sort
sort -r file.txt                   # reverse order
sort -u file.txt                   # sort and remove duplicates
sort -t: -k3 -n /etc/passwd        # sort by field 3, : delim, numeric
sort -k2 -k1 file.txt              # sort by field 2, then field 1

# ── uniq ─────────────────────────────────────────────────────
sort file.txt | uniq               # remove consecutive duplicates
sort file.txt | uniq -c            # count occurrences
sort file.txt | uniq -d            # show only duplicated lines
sort file.txt | uniq -u            # show only unique lines

# ── tr — translate / delete characters ───────────────────────
tr 'a-z' 'A-Z' <<< "hello"        # HELLO (uppercase)
tr -d '\n' < file.txt              # remove all newlines
tr -s ' ' < file.txt               # squeeze repeated spaces to one
echo "a,b,c" | tr ',' '\n'        # comma → newline (split)

# ── diff & column ────────────────────────────────────────────
diff file1.txt file2.txt           # show differences
diff -u file1.txt file2.txt        # unified format (git-style)
column -t -s: /etc/passwd          # align : delimited file into columns
column -t data.csv -s,             # pretty-print CSV

Notes

sed -i '' — macOS requires empty string arg with -i; Linux uses just -i
awk -F: — sets the field separator; $1=first field, $NF=last field, NR=line number
awk END — block runs after all lines are processed; use for totals
sort | uniq -c — the classic frequency count pattern; sort results with sort -rn
tr — works character by character; -d=delete, -s=squeeze repeated chars
column -t — instant aligned table output; great for making CSV readable

Practical One-Liners

Common Mistakes

  • Using sed -i 'backup.bak' on macOS — macOS sed requires a space: sed -i .bak or sed -i ''
  • Using uniq without sort first — uniq only removes adjacent duplicates; unsorted input gives wrong results
  • Forgetting -F in awk for non-space delimiters — default is any whitespace, not commas or colons

Tips

  • Test sed without -i first to preview changes — add -i '' only when the output looks right
  • awk '!seen[$0]++' is the idiomatic way to deduplicate while preserving order — memorize it
  • Combine column -t with less -S to browse wide tabular output without line wrapping
  • Use sd (brew install sd) as a modern sed alternative with cleaner syntax for quick substitutions

Summary

  • sed — line-by-line substitution and deletion; -i '' for in-place on macOS
  • awk — field-based processing; $1…$NF, NR, BEGIN/END
  • sort | uniq -c | sort -rn — the frequency count pipeline
  • cut · tr · column — quick field extraction, char translation, table formatting
