Robust Script Template

#!/usr/bin/env bash
set -euo pipefail              # exit on error, unbound vars, pipe fail
IFS=$'\n\t'                    # safer word splitting

# ── Script metadata ───────────────────────────────────────────
readonly SCRIPT_DIR="$(cd "$(dirname "${BASH_SOURCE[0]}")" && pwd)"
readonly SCRIPT_NAME="$(basename "$0")"
readonly LOG_FILE="/tmp/${SCRIPT_NAME}.log"

# ── Logging helpers ───────────────────────────────────────────
log()   { echo "[$(date '+%H:%M:%S')] $*" | tee -a "$LOG_FILE"; }
info()  { log "INFO  $*"; }
warn()  { log "WARN  $*" >&2; }
error() { log "ERROR $*" >&2; exit 1; }

# ── Cleanup on exit ───────────────────────────────────────────
tmpfile=$(mktemp)
trap 'rm -f "$tmpfile"' EXIT

# ── Argument parsing with getopts ────────────────────────────
usage() { echo "Usage: $0 [-v] [-o output] input"; exit 1; }
verbose=false; output=""
while getopts ":vo:h" opt; do
  case $opt in
    v) verbose=true ;;
    o) output="$OPTARG" ;;
    h) usage ;;
    :) error "Option -$OPTARG requires argument" ;;
    \?) error "Unknown option: -$OPTARG" ;;
  esac
done
shift $(( OPTIND - 1 ))        # remaining args start at $1

# ── Require argument check ────────────────────────────────────
[[ $# -lt 1 ]] && usage
input="${1:?Input file required}"

# ── Color output ─────────────────────────────────────────────
RED='\033[0;31m'; GREEN='\033[0;32m'
YELLOW='\033[1;33m'; NC='\033[0m'  # NC = No Color
echo -e "${GREEN}Success${NC}"
echo -e "${RED}Error${NC}"
echo -e "${YELLOW}Warning${NC}"

# ── Debugging ────────────────────────────────────────────────
set -x                         # print each command before executing
set +x                         # stop tracing
bash -n script.sh              # syntax check without running
bash -x script.sh              # run with trace output

Notes

set -e — exit immediately on any error; prevents cascading failures
set -u — treat unset variables as errors; catches typos in variable names
set -o pipefail — pipeline fails if any command in it fails (not just the last)
${BASH_SOURCE[0]} — reliable way to get the script's own path, even when sourced
getopts — POSIX standard arg parser; handles -v style flags; use OPTARG for values
trap ... EXIT — cleanup always runs whether exit is normal, error, or signal
bash -n — syntax check only; essential before deploying a script

Common Mistakes

  • Omitting set -euo pipefail — scripts silently continue past errors, leaving things in broken state
  • Using $SCRIPT_DIR as just $(dirname $0) — breaks when script is called from another directory; use ${BASH_SOURCE[0]}
  • Hardcoding paths like /usr/local/bin — Apple Silicon Macs use /opt/homebrew/bin; use command -v instead

Tips

  • Add say "Script complete" at the end of long-running scripts — get an audio notification when done
  • Use cmd | pbcopy constantly — pipe any output directly to clipboard for pasting elsewhere
  • Use osascript -e 'display notification "Done" with title "Script"' for macOS notifications from scripts
  • Use shellcheck (brew install shellcheck) to lint scripts — catches dozens of common bash mistakes automatically

Summary

  • set -euo pipefail — the single most important line to add to every script
  • getopts · trap EXIT · mktemp — the three pillars of robust scripts
  • pbcopy · open · say · osascript — macOS power tools for automation
  • shellcheck + bash -n — lint and syntax-check before every deploy
