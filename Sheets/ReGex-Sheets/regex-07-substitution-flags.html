Substitution

Substitution replaces matched text with a replacement string. The replacement can reference captured groups via tokens like $1 / \1 (by number) or $<name> / \g<name> (by name). A callback function (JS) or lambda (Python) can be used as the replacement for dynamic transformations. split() uses a pattern as a delimiter and optionally captures separator groups into the result array.

Key Rules

  • In JS, replace() without the g flag replaces only the first match.
  • replaceAll() (JS ES2021+) always replaces all matches — requires a string or a /g regex.
  • In Python, re.sub() replaces all matches by default; use count= to limit.
  • Replacement token $& (JS) / \g<0> (Python) inserts the entire match.
  • A function replacement receives the full match and all group captures as arguments.

Substitution & Split Examples

// ── JS: replace with tokens ───────────────────────────────────
"John Smith".replace(/(\w+) (\w+)/, "$2, $1")   // "Smith, John"
"2024-01-15".replace(
  /(?\d{4})-(?\d{2})-(?\d{2})/,
  "$/$/$"
)                                                 // "15/01/2024"

// ── JS: replace with function ─────────────────────────────────
"hello world".replace(/\b\w/g, c => c.toUpperCase()) // "Hello World"
"a1b2c3".replace(/\d/g, n => n * 2)                  // "a2b4c6"

// ── JS: split ─────────────────────────────────────────────────
"one, two,  three".split(/,\s*/)  // ["one", "two", "three"]
"2024-01-15".split(/[-\/]/)       // ["2024", "01", "15"]

// ── Python: re.sub ────────────────────────────────────────────
import re
re.sub(r'(\w+) (\w+)', r'\2, \1', 'John Smith') # 'Smith, John'
re.sub(r'\d', lambda m: str(int(m.group())*2), 'a1b2c3') # 'a2b4c6'
re.sub(r'\s+', ' ', 'too   many   spaces')       # 'too many spaces'
re.sub(r'[aeiou]', '*', 'hello', count=2)        # 'h*ll*o' — limit 2

// ── Verbose mode (Python) ─────────────────────────────────────
pattern = re.compile(r"""
  (\d{4})   # year
  -         # separator
  (\d{2})   # month
  -         # separator
  (\d{2})   # day
""", re.VERBOSE)

Notes

$2, $1 — swaps two captured groups in the replacement string
$<name> — named group reference in JS replacement; cleaner than numbered tokens
Function as replacer — receives (match, p1, p2, …, offset, string) in JS
split(/,\s*/) — splits on comma plus any trailing whitespace; handles inconsistent spacing
re.sub(…, count=2) — Python's count parameter limits how many replacements are made
re.VERBOSE — allows whitespace and # comments inside the pattern for readability

Common Mistakes

  • Forgetting the g flag in JS replace() — only the first match is replaced without it.
  • Using $1 tokens in Python's re.sub() — Python uses \1 or \g<1> in the replacement string.
  • Writing a literal $ in a JS replacement string without doubling it — use $$ for a literal dollar sign.
  • Using re.VERBOSE and forgetting to escape spaces that should match — in verbose mode, spaces are ignored.

Tips

  • Use a function replacer in JS for any transformation that can't be expressed with static tokens (case changes, math, lookups).
  • Use re.VERBOSE for long, complex patterns — inline comments make them maintainable.
  • Prefer replaceAll() over replace(/pattern/g) in JS when the pattern is a plain string — it's clearer in intent.

Summary

  • $1 / \1 — insert captured group in replacement (JS / Python).
  • Function replacers allow dynamic substitutions beyond static tokens.
  • split(/pattern/) — use regex as delimiter for flexible tokenization.
  • re.VERBOSE / x flag — add whitespace and comments for complex patterns.
